HTML Images

Displaying Images

The <img> tag is used to display an image on the web page.

The <img> tag does not have an end tag but makes use of the following key attributes:

  • src: the src attribute specifies the location of the image.
  • alt: the alt attribute specifies alternate text to display if the image cannot be displayed.
  • width: the width attribute tells the browser the width of the image.
  • height: the height attribute tells the browser the height of the image.

Image Locations

When the browser goes to find the image source file you want to display, it begins by looking in the same folder as the web page.

It is common to store images in a sub-folder. If this is the case, you must include the folder name in the src attribute.

Where to get Images

Images for your web page may of course be personal images created or taken by yourself. There are also many online images that are available for use. Just be careful of copyright laws.

Some sites that offer free photos are:

  • Pexels.com
  • Unsplash.com
  • Google Images (Use Tools/Usage rights to filter images)

Editing Images

It is often useful to edit images for your web page. Common tasks include resizing images, cropping images, modifying colors, pulling colors from an image, etc.

There are many photo editing tools available, but I recommend paint.net. It's a free download and has many powerful features.
