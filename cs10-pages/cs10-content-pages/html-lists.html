HTML Lists

Ordered & Unordered Lists

Two common types of lists are the ordered list and the unordered list. To create a list simply use the <ul> element for an unordered list and the <ol> element for an ordered list. List items get placed inside of an ordered or unordered list by using <li> elements.

Unordered List Example:

  • List item 1
  • List item 2
  • List item 3
  • List item 4

Ordered List Example:

  1. List item 1
  2. List item 2
  3. List item 3
  4. List item 4
