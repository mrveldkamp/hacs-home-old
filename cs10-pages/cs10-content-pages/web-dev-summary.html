Summary

Concept Checklist

The following list summarizes the knowledge and skills you should have learned throughout the "Intro to Web Development" course.

I can use HTML to create a web page with basic content:

  • HTML Page Structure: <!DOCTYPE html>, <html>, <head>, <title>, <body>
  • Headings and Horizontal Rules: <h1>, <h2>, ..., <h6>, <hr>
  • Paragraphs and Line Breaks: <p>, <br>
  • Comments: <!-- Comment Goes Here -->
  • Images: <img>
  • Links: <a>
  • Lists: <ul>, <ol>, <li>
  • Containers: <strong>, <em>, <span>, <div>

I can style a web page using CSS:

  • CSS How To: external style sheet <link>, CSS Syntax, Selectors, Comments
  • Style Text: color, font-family, font-size, ...
  • Style Links: style different states (link, visited, hover, active)
  • Box Model: border, margin, padding, width, height, centered fixed-width content
  • Backgrounds: background colors / images

I can explain how the web works and publish a web page.

  • Local Development, web hosting, servers, url, etc.
  • Publish a page using Github Pages.

I can use HTML and CSS to create a web page from a given design.
