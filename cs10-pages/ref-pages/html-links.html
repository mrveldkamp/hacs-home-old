HTML Links

Links Syntax

The <a> tag is used to create a link. The href attribute specifies the destination address of the link. The content between the start and end tag is what needs to be clicked in order to activate the link.

Open Link in New Tab

In order to make a link open in a new tab, the target attribute must be set to '_blank'.

Images as Links

An image may be used as a link simply by placing an <img> tag between the start and end tags of the <a> element.
