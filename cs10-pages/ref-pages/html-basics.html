HTML Basics

HTML stands for Hyper Text Markup Language and is used to describe the structure and content of a web page using various markup tags.

HTML Page Structure

Below is an example of a very basic HTML page structure...

  • The <!DOCTYPE html> declaration defines this document as an HTML5 document.
  • <html></html> : The <html> element wraps all the content on the entire page. This element is known as the root element.
  • <head></head> : The <head> element is a place for information about your web page that isn't visible content.
  • <body></body> : The <body> element is a place for the visible content of your web page, such as: headings, paragraphs, lists, images, links, etc.

HTML Title

The <title> element belongs inside of the <head> element. The <title> element contains the title of the web page. This web page title is not displayed in the web page but the web browser will use this title for things such as: the tab heading, search results, etc.
