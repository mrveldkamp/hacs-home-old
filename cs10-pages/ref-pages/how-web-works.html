How the Web Works

What is a Web Page?

A web page consists of a bunch of files that get displayed by a web browser.

A simple web page will usually include the following files:

images folder...

...contains the image files to be used in the web page.

styles.css...

...contains the style rules that control things like color, fonts, layout, etc.

index.html...

...contains the content of a web page and is the document that the web browser will open. This document tells the browser how to put together the content, styles and images for the web page.

Publishing a Web Page

To make a web page visible to anyone with an Internet connection, you must upload the web page files to a server that is connected to the World Wide Web. Anyone can then use the Internet and the web page address to connect to the server and view the web page files in a web browser on their computer.

It is recommended to develop and test web pages on a local computer before making them public. This local development will be the main focus of this course.
