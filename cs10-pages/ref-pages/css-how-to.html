CSS How To

CSS stands for Cascading Style Sheet. CSS is used to select html elements and apply stule rules to these elements. Use CSS to control colors, fonts, borders, margins, etc.

CSS Syntax

Select html elements and apply one or more style rules to the elements.

Select all paragraph elements and make the text 18px and blue.

CSS Where To

There are a number of places where CSS may be placed. I recommend putting CSS code inside of an external style sheet. A separate file is created to store all CSS style rules and the html document will link to this css file.

index.html

styles.css

Using an external style sheet keeps the content and style of the page separate and also allows the reuse of a style sheet on multiple web pages.

CSS Comments

Comments in CSS are done using /* and */.
