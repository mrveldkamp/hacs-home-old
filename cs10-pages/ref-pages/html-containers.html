HTML Containers

Text Containers

The following elements may be used to select parts of text:

  • <strong></strong> : indicates text is important. Usually displays as bold.
  • <em></em> : indicates text is emphasized. Usually displays as italic.
  • <span></span> : selects part of a text without using any default styles.

Example:

In this paragraph: I am very important, I am emphasized and I am in a span... but you can't tell.

The <div> Element

The <div> element may be used for grouping multiple html elements together. A <div> element has no default styles but serves as a block container for other elements. When we learn how to do our own styles you'll see how useful the <div> element can be.
