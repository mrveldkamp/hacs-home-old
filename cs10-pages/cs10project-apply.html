Purpose of Project

Plan and build a web page / program that you are interested in. Demonstrate your ability to confidently use course concepts and explore some new concepts.

You may complete this project individually or with a partner.

What to Create?

Create programs that you are interested in and that are interesting / useful.

  • Plan and create an original idea.
  • Re-create a program that already exists.
  • Modify / extend course examples & assignments.

Concept CheckList

Be sure to use all of the basic concepts listed below in your project. Consider using the extended concepts, where applicable.

Basic Concepts

  • Use HTML and CSS to create a visual display for the program.
  • Use JavaScript to get user input using one or more of the following: text input, numerical input, mouse events, key events, ...
  • Use JavaScript to dynamically update output (content, style, ...)
  • Use variables to manage data.
  • Use if statements to control what code to execute.

General Extended Concepts Learn more...

  • Group related variables with objects.
  • Organize program with functions.
  • Utilize timeouts and intervals.

Web Development Extended Concepts Learn more...

  • Use advanced CSS layout techniques (float, flexbox, grid, ...).
  • Create a multi-page site with easy to use navigation.
  • Implement tabbed content in a single page.
  • CSS Transformations, Transitions and Animations.

Graphics & Animation Extended Concepts Learn more...

  • Control rate of animation(change direction, accelerate, ...)
  • Implement click detection of circles and/or rectangles.
  • Implement collision detection of circles and/or rectangles.
  • Create a user controlled player (mouse & keyboard).
  • Incorporate an animated sprite.
  • Implement multiple levels / stages.
